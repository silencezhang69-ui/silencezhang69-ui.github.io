CURRICULUM

十大成长模块

每一个模块都不是单独的知识点，而是你真正改变的一个维度。

⚡

MODULE 01

认知觉醒

你有没有想过，为什么明明知道要做，却还是拖？为什么努力了半天，感觉原地踏步？这一切的根源，不是懒，是认知。这个模块帮你打破认知失调，重建底层思维逻辑——从此，你不是靠意志力硬撑，而是真正想通了。

⏱

MODULE 02

时间管理

忙了一整天，却感觉什么都没做完？时间管理不是让你多装几个 App，而是重新定义「什么事最值得做」。这个模块教你用一套底层逻辑，把每一天的时间都用在真正推动你成长的事情上。

🧠

MODULE 03

成长思维

真正改变人生的，不是某一次突破，而是每天 1% 的迭代积累。这个模块让你建立成长飞轮——不管遇到挫折、失败还是瓶颈，你都能把它转化为燃料，持续向上。

💬

MODULE 04

沟通高手

你说了很多，对方却完全没听进去——这不是你表达能力差，是你不知道对方真正在意什么。这个模块教你精准读懂人心，让你开口就能说到点上，让每一段关系因为你的到来而变得更好。

✦

MODULE 05

存在力

有些人一进房间，所有人都会看向他；有些人说了半天话，没人记得。差距在哪？在存在力。这个模块系统训练你的气场、边界感与被记住的能力，让你走进任何场合都自带主角光环。

🎯

MODULE 06

领导力

领导力不是职位给的，是你挣来的。为什么有人明明没有头衔，却让所有人心甘情愿跟着走？这个模块让你掌握真正的影响力密码——带人、用人、激发人，在任何团队里成为那个不可或缺的人。

💼

MODULE 07

职场高手

同样的起点，三年后差距天壤之别——差的不是努力，是职场底层规则。这个模块带你提前看懂职场的游戏规则：怎么展示自己、怎么建立信任、怎么让领导和同事都觉得「这个人靠谱」。

🔥

MODULE 08

销售技巧

每个人都在销售——销售自己的想法、能力、价值。这个模块不教你话术，而是教你真正读懂人心：对方在抗拒什么、需要什么、什么时候会说「好」。从此你不用求人，而是让对方主动靠近你。

🎤

MODULE 09

公众演说

一次好的演讲，可以改变一个人对你的全部印象。这个模块不只是教你「怎么讲」，而是教你怎么抓住一群人的注意力、触动他们的情绪、让他们记住你说的话。站上台，就是你的舞台。

💞

MODULE 10

亲密关系

很多关系的破裂，不是因为不爱，而是因为不懂。这个模块帮你真正理解亲密关系的运作逻辑：怎么表达需求不伤人、怎么化解冲突不冷战、怎么让一段关系越走越深——而不是越来越陌生。

PHILOSOPHY

我的带教理念

不是单纯传授知识，而是帮你真正改变认知、内化行动。

🧠

基于心理学的精准判断

运用认知失调、损失厌恶、自我效能感、共情愧疚等心理学理论， 在接触前通过文字与内容判断每位学员的真实抗拒点。

潜意识的反应，才是最真实的。针对每个人，分类记录、精准化解。

🤝

学长学姐视角，而非导师姿态

以学长学姐的角度去分享，耐心陪伴，而不是评判。

用文字反馈学员的优点，让他们看到自己—— 把意识变成潜意识，把道理变成真正属于自己的东西。

💡

四个教练沟通原则

始终保持学长学姐身份，不销售、不施压； 每次打卡给予正向反馈； 多分享自身真实经历； 提高沟通频率与效率——要问问题，学生才会真正开口。
